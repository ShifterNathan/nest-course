# Car dealership 

## Technologies used for this project: 
- Node: https://nodejs.org/en
- Docker: https://docs.docker.com/get-docker/
- TablePlus: https://tableplus.com

## Download and run:

1. Clone the project 
2. Execute:
```
yarn install
```
3. Raise the DB:
```
docker-compose up -d
```
4. Use table plus to connect and view the DB
5. Populate DB:
```
https://localhost:3000/seed
```
